Was ist Programmierung?

Programmierung ist die Kunst, dem Computer Befehle zu geben - so präzise und klar, dass er exakt das tut, was du willst.

Die Analogie: Du schreibst ein Kochrezept für den Computer. Jeder Schritt muss perfekt erklärt sein, sonst versalzt er die Suppe!

Warum ist Programmieren wichtig?

🌍

Die Welt läuft auf Code

Alles, was digital ist, wurde von jemandem programmiert:

  • Instagram-App → Programmiert von tausenden Entwicklern
  • Dein Smartphone-OS → Millionen Zeilen Code
  • Banken-Systeme → Sicherheitscode vom Feinsten
  • Selbstfahrende Autos → Millionen Zeilen Code
💼

Top-Karrierechancen

Programmierer sind die gefragtesten Arbeitskräfte:

  • 💰 Sehr gute Gehälter (oft 60.000€+ pro Jahr)
  • 🌐 Jobs überall auf der Welt (Remote möglich)
  • 🚀 Schnelle Karriereaufstieg
  • 🎯 Vielfältige Bereiche (Spiele, KI, Apps, Web, etc.)
🧠

Denken lernen

Programmieren lehrt dir, wie man Probleme systematisch löst:

  • Problem zerlegen in kleine Schritte
  • Logisch denken und Fehler finden
  • Geduld und Durchhaltevermögen
  • Kreativität und Innovation

Programmiersprachen - Ein Überblick

🟡 Python
Einsteigerfreundlich, vielseitig. Verwendet für: Data Science, KI, Web-Entwicklung, Scripts. Beispiel: Instagram wurde teilweise in Python geschrieben.
🔵 JavaScript
Die Sprache des Web-Browsers. Verwendet für: Interaktive Websites, Web-Apps, Games. Läuft auf JEDEM Computer (im Browser).
☕ Java
Universell, zuverlässig. Verwendet für: Enterprise-Software, Banking-Systeme, Android-Apps. Motto: "Write once, run anywhere"
🟪 C++
Sehr schnell, sehr komplex. Verwendet für: Video-Spiele, Hochleistungs-Software, Betriebssysteme. Beispiel: Unreal Engine
🟦 C#
Modern, von Microsoft. Verwendet für: Windows-Apps, Games (Unity), Web (C# Backend). Ähnlich wie Java, aber neueres Design.
📕 Ruby
Elegant, schnell zu programmieren. Verwendet für: Web-Apps, Startups. Motto: "Programmierer-Glück vor Computer-Geschwindigkeit"
🦀 Rust
Modern, sicher, schnell. Verwendet für: System-Software, Embedded Systems. Die Sprache der Zukunft?

Für Anfänger empfohlen:

🏆 Python - Einfache Syntax, mächtig, überall einsetzbar

🏆 JavaScript - Du siehst sofort Ergebnisse im Browser

🏆 Scratch - Visuelles Programmieren mit Blöcken (für Anfänger)

Grundkonzepte der Programmierung

📦

Variablen - Behälter für Daten

Was: Ein "Behälter", der einen Wert speichert

Analoge: Eine Box mit einer Beschriftung

alter = 15
name = "Max"
höhe = 1.75
🔀

Bedingte Anweisungen (If-Else)

Was: Der Computer trifft eine Entscheidung

Logik: "Wenn X wahr ist, dann mach Y, sonst mach Z"

if alter >= 18:
print("Du darfst Auto fahren")
else:
print("Du musst noch warten")
🔁

Schleifen - Wiederholungen

Was: Ein Befehl, der mehrfach ausgeführt wird

Sinn: Statt 100x die gleiche Zeile zu schreiben, schreibst du 1x und sagst "wiederhole 100x"

for i in range(5):
print("Hallo " + str(i))

Ausgabe: Hallo 0, Hallo 1, Hallo 2, Hallo 3, Hallo 4

📋

Funktionen - Wiederverwendbarer Code

Was: Ein Codeblock, den du mehrmals verwenden kannst

Analoge: Ein Rezept - einmal geschrieben, immer verwendbar

def grüße(name):
return "Hallo " + name

grüße("Max") # Output: "Hallo Max"
grüße("Lisa") # Output: "Hallo Lisa"
📚

Datenstrukturen - Organisieren von Daten

Arrays (Listen): Mehrere Werte in einer Reihe

früchte = ["Apfel", "Banane", "Orange"]
print(früchte[0]) # Output: "Apfel"

Dictionaries (Wörterbücher): Schlüssel-Wert Paare

schüler = {"Name": "Max", "Alter": 15, "Klasse": "4B"}

Pseudocode: Der Plan vor dem Programmieren

Was ist Pseudocode?

Pseudocode ist wie Programmieren in Deutsch - einfach zu verstehen, aber nicht maschinenlesbar.

Du schreibst den Plan auf, bevor du Code schreibst!

Beispiel: Ein einfaches Spiel

PROGRAMM: Zahlenratespiel
1. Denke an eine Zahl zwischen 1 und 100
2. Zeige dem Spieler "Ratiere eine Zahl"
3. Der Spieler gibt eine Zahl ein
4. WENN Spieler-Zahl = Gedachte-Zahl DANN
    Zeige "Gewonnen!"
5. SONST WENN Spieler-Zahl < Gedachte-Zahl DANN
    Zeige "Zu niedrig, versuche nochmal"
    GOTO Schritt 3
6. SONST
    Zeige "Zu hoch, versuche nochmal"
    GOTO Schritt 3

Der erste Schritt: Dein erstes Programm

Python ist perfekt für Anfänger!

Installation: Geh zu python.org und lade Python herunter

Dein erstes Programm:

print("Hallo Welt!")

Ergebnis: "Hallo Welt!" wird auf dem Bildschirm angezeigt!

Etwas Komplexeres:

name = input("Wie heißt du? ")
alter = input("Wie alt bist du? ")
print(f"Hallo {name}, du bist {alter} Jahre alt!")
