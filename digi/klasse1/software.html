Was ist Software?

Software sind die Programme und Daten, die auf deinem Computer laufen. Im Gegensatz zu Hardware kannst du Software nicht anfassen - es sind Anweisungen und Informationen.

Einfach merken: Software = die Befehle, die der Computer ausführt

Arten von Software

🖥️

Betriebssystem (OS)

Funktion: Das "Gehirn" der Software. Es verwaltet alle Hardware-Ressourcen und koordiniert alle anderen Programme.

Beispiele: Windows, macOS, Linux, Android, iOS

Aufgabe: Speicher verwalten, Programme starten, Dateien organisieren

📱

Anwendungssoftware (Apps/Programme)

Funktion: Programme für spezifische Aufgaben.

Beispiele:

  • Microsoft Word, Google Docs (Textverarbeitung)
  • Excel, Google Sheets (Tabellenkalkulation)
  • Firefox, Chrome (Webbrowser)
  • Photoshop, GIMP (Bildbearbeitung)
  • Discord, Telegram (Kommunikation)
⚙️

Systemsoftware & Treiber

Funktion: Programme, die Hardware-Komponenten steuern oder das Betriebssystem unterstützen.

Beispiele: Grafikkartentreiber, Druckertreiber, Systemupdates

🦠

Malware

Warnung! Schädliche Software, die deinem Computer schadet!

Arten:

  • Viren - infizieren andere Dateien
  • Trojaner - verstecken sich wie nützliche Programme
  • Ransomware - sperrt deine Dateien und fordert Geld
  • Spyware - spioniert dich aus

Schutz: Guter Antivirenschutz, keine verdächtigen Downloads

Unterschied: Hardware ↔ Software

Hardware Software
Anfassbar Nicht anfassbar (virtuell)
Physische Komponenten Programme & Daten
Teuer in Anschaffung Oft kostenlos oder lizenzpflichtig
Braucht Zeit zum Kaputtgehen Kann sofort kaputt sein

💡 Merksatz

Hardware = Körper | Software = Gedanken

Ein Computer braucht beide: Hardware für die Verarbeitung und Software für die Anweisungen!
