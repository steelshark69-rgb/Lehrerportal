Kurze Definition

Das Internet ist ein großes Netzwerk von Computern, die weltweit miteinander verbunden sind. Milliarden von Computern sind über Leitungen, Kabel und drahtlose Verbindungen miteinander verbunden.

Wichtige Begriffe

IP-Adresse

Was: Eine eindeutige "Postadresse" für jeden Computer im Internet.

Format: 192.168.0.1 (vier Zahlenblöcke)

Analoge: Wie deine Hausnummer - ohne sie würde niemand wissen, wohin er deine Post senden soll!

Website / Web-Seite

Was: Eine Sammlung von Seiten, die du im Browser ansehen kannst.

Beispiele: www.google.com, www.wikipedia.org, www.youtube.com

Speicherort: Websites werden auf speziellen Computern (Servern) gespeichert

Browser

Was: Ein Programm, mit dem du das Internet "durchsurfst".

Beispiele: Google Chrome, Firefox, Safari, Microsoft Edge

Aufgabe: Hole Websites von Servern und zeige sie dir an

Server

Was: Ein besonderer Computer, der 24/7 läuft und Websites/Dateien speichert.

Aufgabe: Sendet Daten an deine Computer/Handy, wenn du etwas anforderst

Analoge: Wie eine Bibliothek - du fragst an, und sie schickt dir das Buch

URL (Web-Adresse)

Was: Die Adresse einer Website, die du in die Adressleiste tippst.

Beispiel: https://www.google.com

Teile:

  • https:// = Sicherheitsprotokoll
  • www = World Wide Web
  • google.com = Domainname

Internet vs. World Wide Web (WWW)

Internet: Das gesamte globale Netzwerk aller Computer

World Wide Web (WWW): Der Teil des Internets mit Websites und Hyperlinks

Unterschied: Email, Video-Streaming und Spiele nutzen auch das Internet, sind aber nicht unbedingt "Web"

Wie funktioniert das Internet?

Ein Beispiel: Du besuchst www.google.com

  1. Du tippst die Adresse: Du schreibst "google.com" in den Browser
  2. DNS-Anfrage: Der Browser fragt einen speziellen Server: "Welche IP-Adresse hat google.com?"
  3. IP-Adresse wird gefunden: Der DNS-Server antwortet mit der IP-Adresse von Google
  4. Dein Computer verbindet sich: Dein Computer verbindet sich mit Googles Server
  5. Website wird gesendet: Google schickt die Website-Daten zu dir
  6. Browser zeigt Website: Dein Browser kombiniert HTML, CSS und JavaScript und zeigt dir die schöne Website

Technische Grundlagen

Kabelarten

Glasfaser: Sehr schnell, moderne Technologie

Kupferkabel: Älter, aber immer noch üblich

WLAN/WiFi: Drahtlos, bequem aber etwas langsamer

Mobil (4G/5G): Für Handys und Tablets

Geschwindigkeit messen

Einheit: Mbps (Megabits pro Sekunde)

Beispiele:

  • 50 Mbps = Gute Alltagsgeschwindigkeit
  • 100-500 Mbps = Sehr schnell
  • 1000 Mbps = Ultra-schnell (Glasfaser)

💡 Merksätze

Das Internet = Globales Netzwerk von Computern

Website = Seiten auf einem Server

Browser = Dein Zugangstool zum Internet
