⬅ Zurück

🎯 Entdecke dein Betriebssystem!

Diese Aufgaben kannst du auf deinem Computer durchführen. Folge den Anweisungen und finde heraus, wie dein Betriebssystem funktioniert!

🗺️ Deine Missionen

✓ Mission 1: Systemeinstellungen öffnen

  • ☐ Windows: Drücke Win + I oder gehe zu Start → Einstellungen
  • ☐ Mac: Klick auf → Systemeinstellungen
  • ☐ Linux: Öffne Einstellungen aus dem Menü
  • ☐ Schau dir den Computernamen an - wie heißt dein PC?

    • ✓ Mission 2: Festplatte auslesen

  • ☐ Windows: Öffne "Datei-Explorer" → Klick auf "Dieser PC"
  • ☐ Schau dir deine Festplatte an - wie groß ist sie? Wie viel ist frei?
  • ☐ Finde den Ordner "Benutzer" oder "Users"
  • ☐ Schau in deine "Dokumente" oder "Downloads"

    • ✓ Mission 3: Programme erkunden

  • ☐ Windows: Klick auf Start → zeige alle Apps
  • ☐ Wie viele Programme sind installiert? (Zähle mindestens 10!)
  • ☐ Finde: Notepad, Paint, Rechner (oder Mac-Äquivalente)
  • ☐ Öffne die Datenträger-Bereinigung oder einen ähnliches Tool

    • ✓ Mission 4: System-Informationen finden

  • ☐ Finde heraus: Welches OS hast du? (Windows 10/11, macOS, Linux?)
  • ☐ Wie viel RAM hat dein Computer?
  • ☐ Wie schnell ist deine CPU (Prozessor)?
  • ☐ Wann wurde dein Computer das letzte Mal aktualisiert?

    • ✓ Mission 5: Dateien organisieren

  • ☐ Erstelle einen neuen Ordner namens "Meine Schule"
  • ☐ Erstelle darin Unterordner: "Deutsch", "Mathematik", "Englisch"
  • ☐ Erstelle eine Text-Datei und speichere sie in "Deutsch"
  • ☐ Lerne: Wie kopiert man Dateien? (Ctrl+C, Ctrl+V)

    • 🎉 Glückwunsch!

    Du hast dein Betriebssystem erforscht!

    🎓 Was hast du gelernt?
