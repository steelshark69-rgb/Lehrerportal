Was ist das Dateisystem?

Das Dateisystem ist die Art und Weise, wie ein Computer Dateien und Ordner organisiert und speichert. Es ist wie ein großer Aktenschrank für deinen Computer!

Wichtige Begriffe

📄 Datei

Eine Datei ist eine gespeicherte Informationseinheit. Sie enthält Daten wie Text, Bilder, Videos oder Code.

Beispiele: document.docx, photo.jpg, video.mp4, program.exe

Jede Datei hat:

  • Name: z.B. "Mathe-Hausaufgabe"
  • Dateityp/Erweiterung: z.B. ".docx" (Word), ".pdf", ".jpg"
  • Größe: in Byte, KB, MB, GB

📁 Ordner (Verzeichnis)

Ein Ordner ist ein Container, der Dateien und andere Ordner enthält.

Beispiel: Ein Ordner "Schule" kann einen Unterordner "Mathematik" enthalten, der wiederum Dateien wie "Hausaufgaben.docx" enthält.

Vorteile: Organisiert den Computer, macht das Finden von Dateien leicht

🔗 Pfad

Ein Pfad ist die Route zu einer Datei oder einem Ordner.

Beispiel Windows: C:\Benutzer\DeinName\Dokumente\Schule\Mathematik\Hausaufgaben.docx

Beispiel Mac/Linux: /Users/DeinName/Documents/School/Math/homework.docx

Typische Dateisystem-Struktur

C:\ (Festplatte)
├── Windows (Betriebssystem)
├── Program Files (installierte Programme)
├── Users (Benutzerordner)
│ └── DeinName
│ ├── Desktop (was du auf dem Bildschirm siehst)
│ ├── Dokumente (deine Dateien)
│ │ ├── Schule
│ │ │ ├── Mathematik
│ │ │ │ └── Hausaufgaben.docx
│ │ │ └── Deutsch
│ │ │ └── Aufsatz.docx
│ │ └── Fotos
│ │ └── Urlaub2023.jpg
│ ├── Downloads (Dateien aus dem Internet)
│ └── Musik (MP3-Dateien)
└── Weitere Systemordner

Dateitypen und Erweiterungen

Erweiterung Dateityp Beispiele
.docx, .doc Textdokument Briefe, Aufsätze
.pdf PDF-Dokument Schulbücher, Formulare
.xlsx, .xls Tabellenkalkulation Excel, Noten
.jpg, .png Bild Fotos, Screenshots
.mp3, .wav Audiodatei Musik, Podcasts
.mp4, .avi Videodatei Videos, Filme
.exe Programm Installierbare Software
.zip Archiv Mehrere Dateien komprimiert

Tipps für gute Dateiorganisation

  • ✅ Sinnvolle Ordnerstruktur: Nach Fächer, Projekten oder Themen organisieren
  • ✅ Aussagekräftige Namen: "Mathematik_Schularbeit_Mai2026" statt "Datei1"
  • ✅ Keine unnötigen Dateien: Alte Dateien archivieren oder löschen
  • ✅ Regelmäßig Backups machen: Wichtige Dateien sichern!
  • ❌ Nicht zu viel auf dem Desktop: Übersichtlichkeit halten

💡 Merksatz

Ordner = Aktenschrank | Dateien = Dokumente

Gute Ordnung spart Zeit beim Suchen!
