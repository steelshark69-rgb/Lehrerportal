Das OSI 5-Schichten-Modell

Das Internet funktioniert nach einem standardisierten Modell aus 5 Schichten. Jede Schicht hat eine spezifische Aufgabe:

Die 5 Schichten des Internets

💾

1. Physische Schicht (Physical Layer)

Was passiert: Die Hardware - Kabel, WLAN, Kupferleitungen, Glasfasern

Aufgabe: Übertragung von rohen Bits (0 und 1) als elektrische oder optische Signale

Beispiele: Ethernet-Kabel, WLAN-Wellen

🔗

2. Sicherungsschicht (Data Link Layer)

Was passiert: Daten werden in "Frames" verpackt

Aufgabe: Sichere Übertragung zwischen direkt verbundenen Geräten

Beispiel: MAC-Adressen (z.B. 00:1A:2B:3C:4D:5E)

🗺️

3. Vermittlungsschicht (Network Layer)

Was passiert: Routing - der Weg der Daten wird entschieden

Aufgabe: Daten von A zu B über das beste Netzwerk finden

Wichtig: IP-Adressen (z.B. 192.168.1.1)

🤝

4. Transportschicht (Transport Layer)

Was passiert: Kommunikation zwischen Programmen

Aufgabe: Sicherstellen, dass Daten vollständig und korrekt ankommen

Protokolle: TCP (zuverlässig), UDP (schnell)

🎯

5. Anwendungsschicht (Application Layer)

Was passiert: Das, was du siehst!

Aufgabe: Bereitstellung von Diensten für Anwendungen

Beispiele: HTTP (Web), HTTPS (sicheres Web), Email, FTP

So lädt eine Website Schritt für Schritt

Beispiel: Du öffnest www.google.com

  1. Schritt 1: Du gibst die URL in den Browser ein
  2. Schritt 2: Der Browser fragt einen DNS-Server "Welche IP-Adresse hat google.com?"
  3. Schritt 3: Der DNS antwortet, z.B. "142.251.32.46"
  4. Schritt 4: Der Browser sendet einen HTTP-Request an diese IP-Adresse
  5. Schritt 5: Der Server bearbeitet die Anfrage
  6. Schritt 6: Der Server sendet die HTML-Datei zurück
  7. Schritt 7: Der Browser lädt zusätzlich CSS, JavaScript und Bilder
  8. Schritt 8: Der Browser rendert (zeichnet) die Seite auf deinem Bildschirm

💡 Merksätze

Die 5 Schichten: Physisch → Sicherung → Vermittlung → Transport → Anwendung

Analog zu Post: Briefkasten → Briefträger → Route → LKW → Zusteller → du

DNS = Telefonbuch: Es übersetzt Namen (google.com) in Nummern (IP-Adresse)
