Bonus-Level: Gefangen auf SQL Island!

Die Mission: Du hast einen Flugzeugabsturz überlebt und bist auf einer mysteriösen Insel gestrandet. Die Bewohner hier sprechen eine seltsame Sprache: Sie verstehen nur SQL!

Um von dieser Insel zu entkommen, musst du mit den Dörflern kommunizieren, Gegenstände finden (oder stehlen) und Rätsel lösen. Deine einzige Waffe ist das Terminal.

Dein Überlebens-Handbuch (Cheat-Sheet)

Das Spiel (entwickelt von der TU Kaiserslautern) wird dich durch eine spannende Geschichte führen. Bevor du startest, ist hier noch einmal eine kurze Zusammenfassung deiner wichtigsten Werkzeuge.

1. Die Umgebung scannen (SELECT)

Um herauszufinden, was es auf der Insel überhaupt gibt, musst du Tabellen auslesen. Erinnere dich an unsere Basics:

SELECT * FROM bewohner;

Willst du nur bestimmte Personen finden, nutze den Filter (WHERE):

SELECT * FROM bewohner WHERE beruf = 'Schmied';
Neuer Befehl!

2. Dinge verändern (UPDATE)

Auf der Insel musst du manchmal Dinge aktiv verändern. Vielleicht musst du jemandem ein Schwert geben oder deinen eigenen Status ändern. Dafür gibt es den UPDATE-Befehl.

Bauplan: Ändere in der Tabelle X, setze die Spalte Y auf den neuen Wert, aber NUR dort, wo die Bedingung Z zutrifft!

UPDATE bewohner SET status = 'befreit' WHERE name = 'Paul';
Neuer Befehl!

3. Dinge löschen (DELETE)

Wenn du einen Gegenstand wegwirfst oder einen Gegner besiegst, musst du ihn aus der Datenbank löschen. Achtung: Vergiss niemals das WHERE, sonst löschst du die ganze Insel!

DELETE FROM gegenstaende WHERE name = 'kaputtes Schwert';

Bist du bereit für das Abenteuer?

Öffne das Spiel über den Button unten. Das Spiel merkt sich deinen Fortschritt im Browser. Lies dir die Texte der Dorfbewohner genau durch – sie verraten dir immer, wie die Tabellen und Spalten heißen, die du abfragen musst.

Tipp: Achte bei Texten immer auf die einfachen Anführungszeichen ('Schwert')!
