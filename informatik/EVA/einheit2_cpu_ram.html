🧠 Die CPU (Prozessor) - Das Gehirn

Die Central Processing Unit ist das Herzstück. Sie führt alle Berechnungen durch. Ihre Geschwindigkeit wird in Gigahertz (GHz) gemessen. Je mehr Kerne und GHz eine CPU hat, desto schneller kann sie denken.

🗄️ Der RAM (Arbeitsspeicher) - Der Schreibtisch

Der Random Access Memory ist das Kurzzeitgedächtnis. Hier liegen alle Programme, die gerade im Moment geöffnet sind (z.B. deine 50 Google Chrome Tabs). Ist der RAM voll, wird der PC quälend langsam.

💻 Der Flaschenhals-Simulator (Bottleneck)

Eine super-schnelle CPU bringt nichts, wenn der RAM zu klein ist. Finde die richtige Balance für das Spiel "Cyberpunk 2077"!
